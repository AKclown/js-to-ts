## 快捷键配置
- 默认按键配置(window & mac[ctrl -> control])
1. Object转换操作 `ctrl+shift+j` (功能二)
2. Swagger转换操作 `ctrl+shift+k` (功能三)
3. 添加块级注释行 `ctrl+'` (功能四 => /**  */)

- 你也可以去快捷键设置界面进行自定义设置
